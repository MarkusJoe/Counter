# py 版访问次数计数器

> 翻自 [journey-ad](https://github.com/journey-ad/Moe-counter) 的计数器

# 新特性

* 使用了异步Web框架`FastAPI`来提高效率, 所有代码均为异步操作
* 可以自定义显示图片的数量(长度)
* 支持API接口以便自行开发
* 支持了`MySQL`数据库

# 失去的特性

* 暂未支持`MongoDB`
